פרויקט כריית נתונים: סט נתוני סרטים (IMDb \& Wikipedia)
מאת: אביה שפירא



תיאור הפרויקט:
הפרויקט מתמקד באיסוף, איחוד וניקוי נתונים ממקורות שונים ליצירת מסד נתונים מקיף על סרטי קולנוע. התהליך מתחיל בסינון נתונים מסיביים ממאגרי הנתונים הרשמיים של IMDb, וממשיך בהעשרת המידע החסר (כגון תקציב, הכנסות ותקציר) באמצעות Web Scraping מאתר ויקיפדיה.



דרישות מערכת וספריות:
הקוד מבוסס על Python 3 ומשתמש בספריות הבאות:

pandas ו-numpy - לניהול ועיבוד טבלאות הנתונים.

beautifulsoup4 - לחילוץ מידע מדפי HTML (Wikipedia).

requests - לביצוע שאילתות HTTP.

re - לניקוי נתונים מבוסס ביטויים רגולריים (Regex).

gc - לניהול זיכרון אופטימלי בעבודה עם קבצי ענק.



שלבי העבודה העיקריים:

1. עיבוד נתוני IMDb (Big Data)
סינון: חילוץ סרטים (Movies) בלבד, שהופקו עד שנת 2024 ושאורך הריצה שלהם נע בין 60 ל-300 דקות.

דירוגים: איחוד עם טבלת הדירוגים לקבלת ממוצע ציונים ומספר הצבעות.

שחקנים: זיהוי חמשת השחקנים המובילים לכל סרט (Lead Actors) מתוך טבלת ה-Principals תוך שימוש בשיטת Chunking לניהול יעיל של זיכרון ה-RAM.

2. העשרת נתונים מויקיפדיה (Web Scraping)
חלק זה מבוסס על פונקציה המחפשת דפי סרטים בויקיפדיה ומחלצת מתוכם:

Infobox: חילוץ שפה, מדינה, תקציב והכנסות.

Plot: זיהוי פסקאות התקציר תחת כותרות כמו Plot, Synopsis או Summary.

3.ניקוי נתונים מתקדם (המרה למטבע אחיד (מיליוני דולרים  ושפה ראשית בלבד)



מבנה הטבלה הסופית (final\_movies\_perfect\_union.csv):
התוצר הסופי כולל מעל 17,000 שורות ייחודיות עם העמודות הבאות:

Tconst: מזהה IMDb ייחודי.

primaryTitle ו-startYear: שם ושנת יציאה.

Genres: ז'אנרים של הסרט.

AverageRating ו-numVotes: נתוני פופולריות ודירוג.

Budget ו-BoxOffice: נתונים כספיים מנורמלים במיליוני דולרים.

Language ו-Country: שפת מקור ומדינת הפקה.

Plot: תקציר עלילתי.



הוראות הרצה:
ודאו שכל הספריות מותקנות (pip install pandas beautifulsoup4 requests).

הריצו את תאי המחברת לפי הסדר הכרונולוגי.

שימו לב: שלב ה-Web Scraping עשוי לקחת זמן מה (באזור היממה) בשל כמות הבקשות (הקוד מוגדר לעבוד במקטעים של 100 סרטים בכל פעם כדי לשמור על יציבות).
